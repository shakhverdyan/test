Продакшн-версия сайта находится в папке build, версия для разработки - папка dev

Команды: 
npm run start - создает(при необходимости) папку build и запускает watcher
npm run prod - создает продакшн(минифицированную) версию сайта
gulp - запускает watcher с отслеживанием обновления всех компонентов
gulp build - заново компилирует папку build, предварительно удаляя старое содержимое
gulp sass, gulp webP, gulp scrips, gulp svgm gulp html - компиляция отдельных элементов

Добавление флага --prod к команде попутно ведет к минификации файлов (gulp html --prod)
